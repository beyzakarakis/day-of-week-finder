# Day of Week Finder

This Java program determines the day of the week for a given date entered by the user.  
The user provides the year, month, and day, and the program calculates which day of the week it corresponds to.

## How it works
1. The program prompts the user to enter a year, month, and day.
2. It creates a date object using the provided input.
3. The program determines the day of the week using Java's date/time API.
4. The result is displayed to the user (e.g., Monday, Tuesday, etc.).

## Example
**Input:**
Enter year: (e.g., 2012): 2001
Enter month: 1-12: 11
Enter the day of the month: 1-31: 1

**Output:**
Day of the week is Thursday

## Features
- Works with any valid date.
- Handles leap years automatically.

## Requirements
- Java 8 or higher
